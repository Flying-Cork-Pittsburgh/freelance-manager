Contributors: awoods 
Tags: admin, widget
Requires at least: 3.6
Tested up to: 3.6
Stable tag: 0.1

Used by WordPress freelancers to manage client information from your WordPress dashboard.

== Description ==

Assist WordPress Freelancers to effectively manage information about client site. 


== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ 
directory of your WordPress installation and then activate the Plugin from 
Plugins page.
